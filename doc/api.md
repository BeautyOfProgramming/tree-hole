# 3. 去留言
POST /message
 
## 请求示例

http://139.224.14.39:8888/message

|参数名|类型|参数说明|
| ------------- |:-------------:| :-----|
|replyEmail|String|邮箱|
|content|String|留言内容|

```
{
    "replyEmail":"bysocket@aliyun.com",
    "content":"我的人生几经大起大落..."
}
```

## 返回示例
```
12
```
